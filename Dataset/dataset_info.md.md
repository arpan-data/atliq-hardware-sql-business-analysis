# Dataset Information

## Overview

This project uses the **AtliQ Hardware** dataset provided as part of the **Codebasics SQL Bootcamp** for educational and portfolio purposes.

## Database Tables

### Dimension Tables
- dim_customer
- dim_product
- dim_date

### Fact Tables
- fact_sales_monthly
- fact_gross_price
- fact_manufacturing_cost
- fact_pre_invoice_deductions
- fact_post_invoice_deductions
- fact_freight_cost

## Database

- MySQL
- Fiscal Years: 2020–2021

## Purpose

The dataset is used to perform business analysis, including:

- Sales Performance Analysis
- Customer Analysis
- Product Analysis
- Market Analysis
- Growth Analysis
- Pareto Analysis
- ABC Customer Segmentation
- Product Profitability Analysis
